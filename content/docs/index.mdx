# Documentation

This docs area is generated from MDX files in `content/docs`.

## How It Works

- The rail item is configured with `type: "mdx"` and a `folder`.
- The sidebar tree is generated recursively from that folder.
- This page body is rendered from MDX.

## Authoring Notes

Use folders to create grouped navigation. Add `index.mdx` inside a folder when you want that folder to have its own landing page.

## Table Of Contents

The right-side TOC is generated from `##` and `###` headings in each file.
